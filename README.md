# vk
бот для вк с говнокодом

1. для работы нужен node 24
2. заходи в bot.js вставляй токен группы только в 7 строку const VK_TOKEN = ''
3. сохраняй bot.js, заходив папку data кликай на config.json и вставляй айди админа (твой вк айди циферный)
4. запускай через терминал node bot.js, либо через батник start.bat
